 EV Battery Monitoring System

📌 Overview

This project is a low-cost EV battery monitoring system using ESP32.
It monitors **voltage, current, and temperature** and provides real-time alerts.

🚀 Features

⚡ Voltage Monitoring
🔌 Current Monitoring (ACS712)
🌡️ Temperature Monitoring (DS18B20)
📟 LCD Display (I2C)
🔊 Buzzer Alert System

🧰 Components Used

ESP32 Dev Board
ACS712 Current Sensor
Voltage Sensor Module
DS18B20 Temperature Sensor
16x2 LCD (I2C)
Buck Converter (LM2596)
