EV-Battery-Monitoring-System
Low-cost EV battery monitoring system using ESP32 (Voltage, Current, Temperature monitoring with alert system)
EV Battery Monitoring System
Overview
This project monitors voltage, current, and temperature of an EV battery using ESP32.
 Features
- Voltage Monitoring
- Current Monitoring (ACS712)
- Temperature Monitoring (DS18B20)
- LCD Display
- Buzzer Alert System
Components Used
- ESP32
- ACS712 Current Sensor
- DS18B20 Temperature Sensor
- Voltage Sensor Module
- 16x2 LCD (I2C)
- Buck Converter
